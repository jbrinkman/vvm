# vvm

Valkey Version Manager

## Getting Started

1. Clone the repository:

   ```sh
   git clone https://github.com/valkey-io/vvm.git
   cd vvm
   ```

2. Install dependencies:

   ```sh
   go mod tidy
   ```

3. Build the project:

   ```sh
   go build
   ```

4. Run the project:

   ```sh
   ./vvm
   ```

## Configuration

This project uses [Viper](https://github.com/spf13/viper) for configuration management.
